Report Accidents in The UK

Purpose of the project:
The purpose of this project is to investigate, create, and implement a system that will migrate data from a spreadsheet to a relational database system.
The database system will be equipped with suitable analysis tools that can evaluate current performance.
Additionally, a data analysis is conducted on the causes of accidents in the United Kingdom.

Subtasks:
  1. Selecting a cloud solution.
  2. Defining the database system.
  3. Obtaining and wrangling the data.
  4. Interpreting data through data visualization and statistical analysis.
  5. Improving the system using data science and machine learning.
Tools used:
  • PHP, SQL and MySQL
  • R and Rstudio
  • LibreOffice Calc
  • Python
  • Jupyter Notebook
